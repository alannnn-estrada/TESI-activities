## Installation

1. Clone the repository
2. Create a virtual environment

   ```bash
   # On Windows
    python -m venv venv
    # On Linux
    python3 -m venv venv
    ```

3. Activate the virtual environment

   ```bash
   # On Windows
   .\venv\Scripts\activate
   # On Linux
   source venv/bin/activate
   ```

4. Install the dependencies

   ```bash
    pip install pandas
    pip install numpy
    ```

5. Run the project
